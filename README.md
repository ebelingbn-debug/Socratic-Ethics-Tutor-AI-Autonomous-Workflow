# Socratic-Ethics-Tutor-AI-Autonomous-Workflow
An advanced 23-step Socratic reasoning agent built with Botpress + GPT-4o
Overview

This project implements a structured ethical reasoning system using chained autonomous nodes in a Botpress workflow. The agent guides users through complex moral questions using the Socratic method, without giving advice or moral judgments.

Features

23-step ethical analysis pipeline

Multi-node workflow logic (Clarify → Definitions → Assumptions → Evidence → Counterexamples → Frameworks → Meta-ethics → Summary)

Bias, worldview, stakeholder, and cultural-context awareness

Fully modular system prompts per node

Designed for accuracy, neutrality, and consistency

Works for ethics education, counseling support tools, compliance training, and reflection apps

Tech Stack

Botpress 2024

OpenAI GPT-4o

Conversations API / Autonomous Nodes

Custom prompt engineering
